# Water Resources PhD - Portfolio Website

A professional portfolio website for Saeedeh Abedzadeh, PhD student and Research Assistant at Oklahoma State University, specializing in Water Resources Management and Hydrology.

## Features

- Interactive 3D water cycle visualization
- Scroll-driven atmospheric effects
- Research showcase
- Publications and papers
- Contact information

## Tech Stack

- Next.js 14
- React Three Fiber (3D graphics)
- TypeScript
- Tailwind CSS
- Framer Motion

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
